# 🎭 Meme Generator

A modern, responsive web application that displays random memes using the Meme API.

## Features

- 🎲 **Random Meme Generator** - Get a new random meme with each click
- 📱 **Fully Responsive** - Works perfectly on desktop, tablet, and mobile devices
- 🎨 **Modern Design** - Beautiful purple gradient background with smooth animations
- ⬇️ **Download Feature** - Save your favorite memes locally
- ⚡ **Fast & Lightweight** - Pure HTML, CSS, and JavaScript (no heavy frameworks)
- 🚀 **Hosted on GitHub Pages** - Live at your GitHub Pages URL

## How to Use

1. Open the website (hosted on GitHub Pages)
2. View a random meme
3. Click **Next Meme →** to load a new meme
4. Click **Download** to save the current meme to your device

## API Used

This project uses the free [Meme API](https://api.api-ninjas.com/v1/meme) from API Ninjas.

## Tech Stack

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript (ES6+)
- Meme API

## Setup GitHub Pages

To enable GitHub Pages for this repository:

1. Go to your repository settings at `https://github.com/Derick0kip/Goodwill-Repository-/settings`
2. Navigate to **Pages** section
3. Select `main` branch as the source
4. Your site will be published at `https://Derick0kip.github.io/Goodwill-Repository-/`
